# Hello

I work across clinical medicine, research, and software engineering. I'm particularly interested in trauma surgery, bone infections, and the practical challenges of managing complex cases.

Alongside this, my research work has pushed me toward programming. Good tooling can fundamentally change how large datasets are handled and understood, so I’ve been building a stronger foundation in software development to support my future work.

## Programming

I build tools primarily in Rust, focusing on systems-level problems with an emphasis on security and reliability.

- [**Memfd-Secret**](https://github.com/ZorroTheRelentless/memfd_secret)  
  A Rust crate wrapping the Linux `memfd_secret` syscall to securely store sensitive data in memory. The underlying memory is zeroed when the secret is dropped, and the crate includes full test coverage with property testing.

- [**Shumai**](https://github.com/ZorroTheRelentless/shumai)  
  A Linux CLI tool that automatically renews SSL certificates issued by Porkbun and exposes a health server. Built on top my memfd-secret crate.

- [**This Website**](https://github.com/ZorroTheRelentless/astro-blog)  
  A static site built with Astro from the ground up, without third-party themes. It features a custom layout system, custom multilingual routing, and a focus on clean typography and performance.
