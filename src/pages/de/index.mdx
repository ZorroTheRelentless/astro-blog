## Programmierung

Ich entwickle hauptsächlich Tools in Rust und konzentriere mich dabei auf systemnahe Fragestellungen mit Fokus auf Sicherheit und Zuverlässigkeit.

- [**Memfd-Secret**](https://github.com/ZorroTheRelentless/memfd_secret)
  Eine Rust-Bibliothek, die den Linux ***memfd_secret*** Syscall kapselt, um sensible Daten sicher im Speicher zu verwalten. Der zugrunde liegende Speicher wird beim Freigeben automatisch überschrieben. Das Projekt verfügt über vollständige Testabdeckung einschließlich Property-Testing.

- [**Shumai**](https://github.com/ZorroTheRelentless/shumai)  
  Ein Linux-CLI-Tool, das automatisch SSL-Zertifikate von Porkbun erneuert und einen Health-Server bereitstellt. Es basiert auf meiner memfd-secret-Bibliothek.

- [**Diese Website**](https://github.com/ZorroTheRelentless/astro-blog)  
  Eine statische Website, die von Grund auf mit Astro entwickelt wurde, ohne Drittanbieter-Themes. Der Fokus liegt auf Baushaus, Performance und vollständiger Kontrolle über Layout, Styling und Internationalisierung.
